## afrog-GUI 一个简单的图形化工具
#### 基于AI 用于 Afrog poc 的生成 管理 和测试，构建自己的漏洞库

#### 实现
**afrog poc (AI 生成 ) ——————>资产搜索(获取验证目标)——————>漏洞验证（验证是否可用）————>POC管理**

**完美闭环！**
